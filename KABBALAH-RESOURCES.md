## Scraping Strategies by Source

### Strategy 1: Otzaria Library (HIGHEST PRIORITY -- EASIEST)

```
Difficulty: EASY
Format: Plain text files (TXT)
Legal: Open source / Unlicense
```

**Steps:**
1. Download the latest release from https://github.com/Sivan22/otzaria-library/releases

3. Navigate the folder structure to find Kabbalah/Chassidut texts
4. Files are already in TXT format -- direct ingest possible
5. Cross-reference against our Sefaria-Export to identify unique texts
6. The library includes texts from Dicta (which has content not on Sefaria)

**What to expect:** The Otzaria library aggregates texts from multiple sources. Many texts we're missing from Sefaria may already be here in plain text. This is the single most efficient path to filling gaps.

### Strategy 2: Sefaria API / MCP (HIGH PRIORITY)

```
Difficulty: EASY
Format: JSON (structured)
Legal: CC licenses (varies by text)
```

**Steps:**
1. Use Sefaria API: `https://www.sefaria.org/api/texts/{ref}`
2. Or use the official Sefaria MCP server
3. Verify all texts in Musar + Jewish Thought sections are captured
4. Many "missing" Kabbalah texts are actually in other Sefaria sections

**Key API endpoints:**
- `GET /api/texts/{ref}` -- Get text content
- `GET /api/index/` -- Get table of contents
- `GET /api/search/` -- Full text search
- `GET /api/links/{ref}` -- Get intertextual connections

### Strategy 3: HebrewBooks.org PDF Pipeline (MEDIUM PRIORITY)

```
Difficulty: HARD (requires OCR pipeline)
Format: PDF (scanned images)
Legal: Free access; public domain texts
```

**Steps:**
1. Identify target books by HebrewBooks ID (see confirmed IDs above)
2. Download PDFs: `https://hebrewbooks.org/pdfpager.aspx?req={ID}&pgnum=1`
3. Run through OCR pipeline:
   - Option A: Dicta OCR (https://dicta.org.il) -- best for rabbinic typefaces including Rashi script
   - Option B: Google Cloud Vision API with Hebrew language pack
   - Option C: Tesseract OCR with Hebrew trained data
4. Post-process: abbreviation expansion (use Dicta tools), text cleanup
5. Validate against known passages

**HebrewBooks PDF Download Pattern:**
```
Base URL: https://hebrewbooks.org/pdfpager.aspx?req={BOOK_ID}&pgnum={PAGE_NUM}
Full PDF: https://hebrewbooks.org/{BOOK_ID} (redirects to first page)
```

**Important Notes:**
- HebrewBooks does NOT have a public API
- Bulk downloading should be done respectfully (rate limiting, off-peak hours)
- Contact dev@hebrewbooks.org for any development inquiries
- Many PDFs already have an OCR-searchable text layer

## Technical Notes

### OCR Pipeline for HebrewBooks

**Recommended Stack:**
1. **Dicta OCR** (https://dicta.org.il) -- purpose-built for rabbinic typefaces
2. **Dicta Nakdan** -- add nikud/vocalization after OCR
3. **Dicta Abbreviation Expander** -- expand rabbinic abbreviations
4. **Dicta Scriptural Allusion Finder** -- identify embedded biblical quotes

**Alternative OCR Options:**
- Google Cloud Vision API (Hebrew language model)
- Tesseract OCR with Hebrew training data
- Azure Cognitive Services OCR

### Text Format Standards

All ingested texts should be normalized to:
- UTF-8 encoding
- JSON format matching Sefaria-Export structure
- Structured by chapter/section/paragraph
- Metadata: title, author, date, source, license
- Cross-references to related texts

### Deduplication Strategy

When merging texts from multiple sources (Sefaria-Export, Otzaria, HebrewBooks OCR):
1. Use Sefaria-Export as the canonical version when available
2. Use Otzaria as secondary source (may have different editions)
3. Use HebrewBooks OCR only for texts not available elsewhere
4. Run text comparison to identify edition differences
5. Store edition variants as metadata, not duplicates
